Host på Github

so you want to learn to host

Github

så du har kommet til denne nettsiden fordi du har lyst til å hoste nettsiden din for å gjore det skal vi bruke Github fordi det er utrolig enkelt ok det eneste du trenger er filen din og en github bruker.

det første steget er å lage et repository med navnet (ditt brukernavn).github.io

når du har gjordt det så lager du en tom fil det er her du skal git clone

nå trenger du å åpne en terminal for å legge nettsiden ut

gå til hvor du vil du har lyst til at prosektet skal være å så git cloner du det

nå burde det ligge en mappe som hetter (brukernavn).github.io og inne i den ligger alle filenen dine i

nå burde siden din være på Github bare skriv in (Brukernavn).github.io og den vil komme opp

Hva med Heroku

et veldig populært alternativ er Heroku som da vil hoste nettsiden din for deg men jeg mener at heroku er veldig dårlig å starte med. Heroku er veldig fin fordi den støtter flere ting som Node.js og Python. det er veldig fint fordi det gir deg muligheter lengre nede men med alle mulighetene kan det være vanneskelig å finne hjelp vis et problem dukker opp. heroku kan gjøre veldig mye som Github ikke kan men som din første side så er Github det beste fordi Github i seg selv er en nettside som du kommer til å bruke uansett så det hjelper deg med å bli kjent med det verktøyet også
